# UiNavKit

UiNavKit is the optional in-game tooling companion for `UiNav`.

It hosts the developer-facing inspectors and utilities:
- `General`
- `Selector`
- `ManiaLink UI`
- `ControlTree UI`
- `ManiaLink Builder`
- `ManiaLink Browser`
- `Diagnostics`

`UiNav` remains the stable runtime/dependency plugin.
`UiNavKit` depends on `UiNav`, exports nothing, and is intended for development workflows rather than downstream automation dependencies.
